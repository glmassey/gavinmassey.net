How to Set Up VyOS on VMware Workstation Pro

1. Download VyOS ISO

  1. Visit VyOS Download Page: Go to VyOS download page.
  2. Select Version: Choose the latest stable release or a version that suits your needs.
  3. Download ISO: Click on the download link for the ISO image and save it to your computer.

2. Create a New Virtual Machine in VMware

  1. Open VMware Workstation Pro: Launch the VMware Workstation Pro application.
  2. Create New Virtual Machine: Select “Create a New Virtual Machine” or use “File” > “New Virtual Machine.”
  3. Installation Method: Choose “Installer disc image file (iso)” and browse to the VyOS ISO file.
  4. Guest Operating System: Select “Linux” and “Other Linux 3.x kernel 64-bit.”
  5. Name and Location: Name your VM and choose a storage location for VM files.
  6. Disk Capacity: Allocate disk space (8 GB is typically sufficient) and choose single or multiple file storage.
  7. Customize Hardware: Adjust RAM (512 MB is usually enough) and CPU settings based on your system’s capabilities.
  8. Complete Setup: Review settings and complete the VM creation.

3. Install VyOS

  1. Start VM: Power on the VM, and it should boot from the ISO.
  2. Install VyOS: Log in (default vyos/vyos), type install image, and follow prompts to install.
  3. Reboot and Eject ISO: After installation, reboot the VM and disconnect the ISO from the virtual drive.

4. Configure Network Settings in VyOS

  1. Log into VyOS: Use the credentials vyos/vyos.

  2. Enter Configuration Mode: Type configure.

  3. Configure Interfaces: For example:

    set interfaces ethernet eth0 address '10.0.0.1/30'
set interfaces ethernet eth1 address '192.168.10.1/24'

  4. Commit and Save: Type commit and save.

5. Install VMware Tools on VyOS

  1. Ensure Internet Access: Adjust network settings if necessary for internet access.

  2. Update and Install:

    sudo apt-get update
sudo apt-get install open-vm-tools

  3. Reboot VM: After installation, reboot the VM.

6. Change Default Username and Password

  1. Log into VyOS: Use vyos/vyos to access the CLI.

  2. Enter Configuration Mode: Type configure.

  3. Change Password:

    set system login user vyos authentication plaintext-password 'YourNewPassword'

  4. (Optional) Create New User:

    set system login user newusername authentication plaintext-password 'NewUserPassword'
set system login user newusername level admin

  5. Commit and Save: Use commit and save.

  6. Exit and Test: Log out and test the new credentials.

7. Finalize Configuration

  • Additional Configurations: Set up routing, firewall rules, NAT, etc., as needed.

Notes

  • Ensure you use a strong and secure password.
  • Document the new credentials securely.
  • Test new user access and privileges before discontinuing the default account.
  • Adjust firewall and network settings based on your specific requirements and security policies.

WIM, CIM Guide

1. Retrieve Basic Operating System Information

This command fetches basic details about the operating system.

Get-CimInstance -ClassName CIM_OperatingSystem

2. List All Installed Services

Retrieve a list of all services installed on the machine.

Get-CimInstance -ClassName Win32_Service

3. Get System BIOS Information

Displays details about the system’s BIOS.

Get-CimInstance -ClassName Win32_BIOS

4. List All Running Processes

Shows all currently running processes on the machine.

Get-CimInstance -ClassName Win32_Process

5. Check Disk Space

Get details about logical disks, including free and used space.

Get-CimInstance -ClassName Win32_LogicalDisk

6. Retrieve Computer System Information

Obtain details like computer name, manufacturer, and model.

Get-CimInstance -ClassName Win32_ComputerSystem

7. List Network Adapters and Status

Displays details about the system’s network adapters.

Get-CimInstance -ClassName Win32_NetworkAdapter | Where-Object { $_.NetConnectionStatus -ne $null }

8. Fetch User Account Information

Retrieve a list of user accounts on the system.

Get-CimInstance -ClassName Win32_UserAccount

9. Retrieve Details about System Memory

Get detailed information about physical memory (RAM) modules.

Get-CimInstance -ClassName Win32_PhysicalMemory

10. Check Uptime

Find out how long the system has been up since the last reboot.

(Get-CimInstance -ClassName Win32_OperatingSystem).LastBootUpTime

11. List Installed Software

Shows all software installed on the system.

Get-CimInstance -ClassName Win32_Product

12. Start a Remote Service

Start a service on a remote computer. In this case, we’re starting the “wuauserv” service (Windows Update service) on a remote machine named “RemotePC”.

Invoke-CimMethod -ClassName Win32_Service -MethodName StartService -Filter "Name='wuauserv'" -ComputerName "RemotePC"

13. Get System Boot Configuration

Displays details about the system’s boot configuration.

Get-CimInstance -ClassName Win32_BootConfiguration

14. List All Printers

Retrieve a list of all printers connected or networked to the system.

Get-CimInstance -ClassName Win32_Printer

15. Get Details of a Specific Process

Obtain details about a specific process by its name. Here, we’re fetching details about “notepad.exe”.

Get-CimInstance -ClassName Win32_Process | Where-Object { $_.Name -eq 'notepad.exe' }

16. Set a Static IP Address

Configure a network adapter to use a static IP address. Note: Change the InterfaceAlias and IP details accordingly.

New-CimInstance -ClassName Win32_NetworkAdapterConfiguration -Property @{
    InterfaceAlias = 'Ethernet0'
    IPAddress      = '192.168.1.100'
    SubnetMask     = '255.255.255.0'
    DefaultGateway = '192.168.1.1'
}

17. List All Startup Commands

Retrieve a list of all startup commands set to run when the system starts.

Get-CimInstance -ClassName Win32_StartupCommand

18. Retrieve Environment Variables

Displays all environment variables on the system.

Get-CimInstance -ClassName Win32_Environment

19. Get Detailed Battery Information

For laptops, this command displays details about the battery, including its status and estimated run time.

Get-CimInstance -ClassName Win32_Battery

20. Retrieve All Network Protocol Bindings

Displays all network protocol bindings on the system.

Get-CimInstance -ClassName Win32_NetworkProtocol

Steps to Add "Open PowerShell as Administrator" to Context Menu

Update Registry

  1. Open Registry Editor:

    • Press Win + R to open the Run dialog.
    • Type regedit and press Enter.
    • If prompted by User Account Control (UAC), click Yes to continue.

  2. Navigate to the Context Menu Key:

    • In the Registry Editor, navigate to the following key:

      HKEY_CLASSES_ROOT\Directory\Background\shell

    • You can copy and paste this path into the Registry Editor’s address bar.

  3. Create a New Key for the PowerShell Prompt:

    • Right-click on shell, select New > Key, and name it Open PowerShell here as Administrator.

  4. Set Command to Open PowerShell as Administrator:

    • With the new key (Open PowerShell here as Administrator) selected, right-click and create another new key named command.

    • Select the command key, and in the right pane, double-click on the (Default) value.

    • Set the Value data to:

      powershell -windowstyle hidden -Command "Start-Process powershell -ArgumentList '-NoExit', '-Command', 'cd ''%V''' -Verb RunAs"

    • Click OK.

  5. Optional: Add an Icon:

    • To add an icon to your new context menu item, select the Open PowerShell here as Administrator key.
    • In the right pane, right-click and choose New > String Value.
    • Name it Icon.
    • Double-click on it and set the Value data to powershell.exe.
    • Click OK.

  6. Close the Registry Editor and Restart your computer or log off and back on to apply the changes.

After these steps, when you right-click in the blank space of a folder or on the folder itself in Windows Explorer, you should see the option to “Open PowerShell here as Administrator.”

Important

  • Be cautious while editing the registry. Incorrect changes can cause system issues.
  • If you’re not comfortable making these changes, it’s advisable not to proceed.

Cisco Port Security

Introduction

Cisco Port Security is a feature on Cisco switches that enhances the security of the network by controlling access to switch ports. It limits the devices that can connect to a switch port, thus preventing unauthorized network access.

Use Cases for Cisco Port Security

  • Preventing Unauthorized Network Access: Restricts access to a network by allowing only known MAC addresses to connect.
  • Protecting Against MAC Address Spoofing: Prevents potential attackers from impersonating other devices on the network.
  • Managing Network Access: Controls which devices are allowed to connect to certain parts of the network.

Theory of Cisco Port Security

Port security works by limiting the MAC addresses that are allowed to communicate on a particular switch port. When a port is secured, only devices with specific MAC addresses can access the network through that port. Key aspects include:

  • Secure MAC Addresses: These can be either statically defined or dynamically learned by the switch.
  • Security Violation Modes: Determines the action when a security violation occurs. Options include Protect, Restrict, and Shutdown.
  • Maximum MAC Addresses: Sets the number of MAC addresses allowed on a port.

Implementing Cisco Port Security

Here are the basic steps to configure port security on a Cisco switch:

  1. Enable Port Security:

    interface [interface_type/number]
switchport mode access
switchport port-security

    This sets the switch port to access mode and enables port security.

  2. Specify Secure MAC Addresses:

    • Statically:

      switchport port-security mac-address [mac_address]

    • Dynamically:

      Enable port security and set the maximum number of MAC addresses. The switch will learn the MAC addresses dynamically.

      switchport port-security
switchport port-security maximum [number]

    • Sticky MAC Addresses (Optional):

      For the switch to remember dynamically learned MAC addresses.

      switchport port-security mac-address sticky

  3. Configure Violation Modes:

    switchport port-security violation {protect | restrict | shutdown}

  4. Verify Port Security Configuration:

    show port-security interface [interface_type/number]

Conclusion

Cisco Port Security is an essential feature for securing network access on a switch. By properly implementing port security, network administrators can effectively control physical access to the network, enhancing the overall security posture.

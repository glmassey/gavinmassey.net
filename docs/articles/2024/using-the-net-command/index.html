Using the 'net' command in Windows

Introduction

The net command in Windows is a versatile utility used for a wide array of network and system management tasks. This command-line tool can manage user accounts, network resources, services, and more, making it an essential tool for administrators. This guide delves into the use of the net command with detailed examples and background information on each functionality.

Managing User Accounts and Groups

  1. Create a New User:

    net user Emily StrongP@ssword123 /add

    This command creates a new user named Emily with a specified password.

  2. Delete a User:

    net user Emily /delete

    Removes Emily from the system’s user accounts.

  3. Add a User to a Local Group:

    net localgroup Administrators Emily /add

    Adds the user Emily to the local group ‘Administrators’, granting her administrative privileges.

Managing Network Resources

  1. Create a Shared Folder:

    net share SharedDocs=C:\Documents

    Shares the folder located at C:\Documents under the share name ‘SharedDocs’.

  2. Remove a Shared Folder:

    net share SharedDocs /delete

    Stops sharing the ‘SharedDocs’ folder.

  3. Map a Network Drive:

    net use X: \\ServerName\SharedDocs

    Maps the shared folder ‘SharedDocs’ on ‘ServerName’ to the drive letter X:.

  4. Disconnect a Mapped Drive:

    net use X: /delete

    Disconnects the mapped drive X:.

Controlling Services

  1. Start a Service:

    net start Audiosrv

    Starts the Windows Audio service.

  2. Stop a Service:

    net stop Audiosrv

    Stops the Windows Audio service.

Managing Network Sessions

  1. List Network Sessions:

    net session

    Shows all active sessions connected to the computer.

  2. Disconnect a Network Session:

    net session \\192.168.1.25 /delete

    Ends the session with the computer at IP address 192.168.1.25.

Configuring Network Protocols

  1. Configure Workstation Service:

    net config workstation

    Displays settings and status of the Workstation service, handling client network connections to servers.

  2. Configure Server Service:

    net config server

    Shows configuration of the Server service, which manages this computer’s shared resources.

Displaying Network Statistics

  1. Display Statistics for Workstation Service:

    net statistics workstation

    Reports statistics on the client-side network connections made by this computer.

  2. Display Statistics for Server Service:

    net statistics server

    Provides data on the use of shared resources this computer provides to others.

Managing Workstation and Domain Settings

  1. Manage User Account and Password Settings:

    net accounts

    Reviews or changes settings related to user accounts and passwords.

  2. Add a Computer to Domain Database:

    net computer \\NewPC /add

    Registers ‘NewPC’ in the domain’s computer list.

  3. Remove a Computer from Domain Database:

    net computer \\OldPC /delete

    Removes ‘OldPC’ from the domain’s computer database.

Conclusion

The net command in Windows is invaluable for network and system administrators, offering a wide range of functions for managing various aspects of Windows networks and systems. Its versatility and comprehensive scope make it a key tool for effective network management.

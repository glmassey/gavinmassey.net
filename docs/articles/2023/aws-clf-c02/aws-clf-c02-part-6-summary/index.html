Cloud Concepts

  • Cost-effective and scalable with a pay-as-you-go pricing model.
  • Global infrastructure offering high availability and speed of deployment.
  • Focus on high availability, elasticity, and agility.
  • Operational excellence, security, reliability, performance efficiency, cost optimization, and sustainability are pillars of the AWS Well-Architected Framework.
  • Cloud migration strategies include re-hosting, re-platforming, refactoring, and using AWS Snowball for data transfer.
  • Cloud economics emphasize shifting from CapEx to OpEx, understanding licensing strategies like BYOL, and the importance of rightsizing.

Security and Compliance

  • AWS is responsible for the cloud; customers are responsible for security in the cloud.
  • AWS provides encryption, governance tools, and security log capture.
  • IAM for managing access, importance of securing the root account, and adhering to the principle of least privilege.
  • AWS security services include Amazon Inspector, AWS Security Hub, Amazon GuardDuty, and AWS Shield.

Cloud Technology and Services

  • Deployment options include AWS Management Console, CLI, APIs, and SDKs.
  • Regions, Availability Zones, and edge locations support high availability and low latency.
  • Compute services like AWS Lambda offer serverless options; EC2 provides scalable virtual servers.
  • Amazon RDS offers managed relational databases; Amazon DynamoDB for NoSQL; Amazon Aurora for high-performance database needs.
  • VPC for network isolation; Route 53 for DNS; CloudFront and Global Accelerator for content delivery.
  • S3 for object storage with various classes; EBS for block storage; EFS and FSx for file storage; AWS Backup for backup solutions.

Billing, Pricing, and Support

  • Compute options include On-Demand, Reserved, Spot Instances, Savings Plans.
  • Data transfer charges apply for data movement between regions or out of AWS.
  • Storage options include Amazon S3, EBS, and Glacier, each with different tiers.
  • AWS Budgets and Cost Explorer assist in cost management; AWS Billing Conductor for customized billing.
  • AWS Support plans vary from Basic to Enterprise, offering different levels of support.

Appendix: Technologies and Concepts

  • Understanding of APIs, AWS global infrastructure, machine learning services, and network services.
  • Familiarity with AWS CAF, AWS Compliance, AWS Pricing Calculator, and AWS Professional Services.

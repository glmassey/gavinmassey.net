How to Set Up pfSense on VMware Workstation Pro

In a virtual lab environment, often you will need a network device to accurately reflect a real-world environment. pfSense is a good option and is easy to use.

1. Download pfSense Image

  1. Visit pfSense Download Page: Go to the pfSense download page.
  2. Select Version and Architecture: Choose the latest stable release and select the architecture that matches your system (most likely AMD64).
  3. Select Installer Type: Choose the installer type. For VMware, the CD Image (ISO) installer is typically used.
  4. Download Image: Click on the download link for the ISO image and save it to your computer.

2. Create a New Virtual Machine in VMware

  1. Open VMware Workstation Pro: Launch the application.
  2. Create New Virtual Machine: Select “Create a New Virtual Machine” or use “File” > “New Virtual Machine.”
  3. Installation Method: Choose “Installer disc image file (iso)” and browse to the pfSense ISO file.
  4. Guest Operating System: Select “Other” and then “FreeBSD” (version 14).
  5. Name and Location: Name your VM and choose a storage location for VM files.
  6. Disk Capacity: Allocate disk space (10-20 GB should be sufficient) and choose single or multiple file storage.
  7. Customize Hardware: Adjust RAM (at least 512 MB) and CPU settings based on your system’s capabilities. Also, add at least two network adapters (for WAN and LAN).
  8. Complete Setup: Review settings and complete the VM creation.

3. Install pfSense

  1. Start VM: Power on the VM, and it should boot from the ISO.
  2. Install pfSense: Follow the on-screen instructions to install pfSense. The process is straightforward, with prompts guiding you through.
  3. Reboot and Eject ISO: After installation, reboot the VM and disconnect the ISO from the virtual drive.

4. Initial Configuration of pfSense

  1. Access pfSense Web Interface: Once pfSense has booted, it will display the LAN IP address in the console. Use a web browser from a VM or computer on the same network to access the pfSense web interface.
  2. Run Setup Wizard: The first time you access the web interface, you’ll be prompted to run the setup wizard.
  3. Configure General Information: Set hostname, domain, and primary DNS servers.
  4. Configure WAN Interface: Set up the WAN interface with the appropriate settings (DHCP, static IP, PPPoE, etc.).
  5. Configure LAN Interface: Set the LAN IP address and subnet mask.
  6. Set Admin Password: Change the admin password from the default to a strong, secure password.
  7. Reload Configuration: After completing the setup wizard, pfSense will apply the configuration and reload.

5. Additional Configurations

  • Configure Firewall and NAT Rules: Set up firewall rules to control traffic between networks.
  • Set Up DHCP Server: If needed, configure the DHCP server for the LAN interface.
  • Create VPNs, VLANs, etc.: Depending on your requirements, set up additional features like VPNs, VLANs, traffic shaping, etc.

6. Install VMware Tools on pfSense

  1. Access pfSense Shell: From the pfSense console menu, access the shell (option 8).

  2. Install VMware Tools: Run the following commands:

    pkg update
pkg install open-vm-tools

  3. Reboot pfSense: After installation, reboot pfSense.

7. Finalize Configuration

  • Review Settings: Go through the settings to ensure everything is configured as per your requirements.
  • Backup Configuration: It’s a good practice to backup your pfSense configuration once everything is set up.

Notes

  • Always use a strong and unique password for the admin account.
  • Regularly update pfSense to the latest version for security and stability.
  • Document your configuration settings for future reference.
  • Adjust firewall rules according to your network’s security policy and requirements.
